В ВТБ предложили сделать Python первым «иностранным» языком для
россиян

Язык программирования Python должен стать первым «иностранным» для большинства россиян. С
таким предложением выступил главный экономист по России и СНГ «ВТБ Капитал»
Александр Исаков на ПМЭФ.«Python [высокоуровневый язык программирования] должен
стать первым иностранным языком для большинства наших граждан», — сказал Исаков (цитата по
