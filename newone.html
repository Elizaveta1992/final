Проверка орфографии в Figma

Доступно с обновления за декабрь 2022
Проверка орфографии выполняется для активного текстового поля в режиме редактирования. Как
это работает
-Если обнаружена ошибка, текст подчеркивается красным.
-При открытии контекстного меню можно выбрать один из предложенных вариантов
исправления.
В этом меню также можно добавить слово в словарь и включить/выключить проверку, а также
выбрать язык (есть вариант автоопределения).
Доступные языки
По умолчанию для клиента (десктопной версии) доступна работа со всеми языками,
установленными в системе. Для веб-версии по умолчанию есть проверка английского и
русского языков. Параметры проверки орфографии, заданные в веб-приложении, не применяются к настольному
приложению и наоборот. Если вы переключаетесь между двумя приложениями, вы должны снова
настроить параметры проверки орфографии.
Как включить проверку орфографии
Чтобы настроить автокоррект перейдите (находясь в редакторе) в меню Maim menu (с логотипом
Фигмы), откройте Text и выберите Spell Check.
Чтобы включить проверку орфографии, проставьте чекбокс у пункта Check Spelling.
